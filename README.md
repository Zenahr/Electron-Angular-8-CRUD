# SQLite + Electron + Express + Angular 8 DBMS
SQLite + Electron + Express + Angular 8 DBMS

Basic CRUD.

## Version
SQLite3 (4.0.9) 

Express (4.17.1) 

Angular (8.1.1)

Node.js (10.16.0)

Electron (5.0.7)

## Install
`npm install`

`./node_modules/.bin/electron-rebuild  -f -w sqlite3`


##  Run 
`npm run start:electron`
